<h2 id="project-tech">Technology</h2>

At its core, TheLatest.io leverages the Model-View-Controller paradigm. It achieves this using the following technologies:

- Java Spring Boot
- Spring Data JPA
- MySQL
- Thymeleaf
- BCrypt
